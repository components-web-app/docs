COVER:
- What AbstractComponent provides out of the box (don't re-implement these):
    - IRI / UUID identity
    - uiComponent: string — tells the front-end which Vue component to render
    - uiClassNames: string[] — CSS class sets an admin can apply
    - componentGroups / componentPositions associations
    - _metadata envelope in serialized output

- Step 1: Create the entity extending AbstractComponent
    namespace App\Entity;
    use Silverback\ApiComponentsBundle\Entity\Core\AbstractComponent;

    class Title extends AbstractComponent {
        public ?string $title = null;
    }

- Step 2: Add API Platform resource + Doctrine mapping
    #[ORM\Entity]
    #[ApiResource]
    class Title extends AbstractComponent {
        #[ORM\Column(type: 'text', nullable: true)]
        public ?string $title = null;
    }

- Step 3: Enable Mercure for real-time updates
    #[ApiResource(mercure: true)]
    — what this does: publishes an update to the Mercure hub whenever the resource changes

- Step 4: Run migrations
    bin/console doctrine:migrations:diff
    bin/console doctrine:migrations:migrate
    — review the generated migration before running

- The /component/ URL prefix convention:
    - AbstractComponent automatically prefixes all endpoints with /component/
    - e.g. GET /component/titles, POST /component/titles, GET /component/titles/{id}
    - Custom prefix: set routePrefix on #[ApiResource]

- Naming conventions:
    - PHP class name Title → uiComponent value "Title" → Vue component CwaComponentTitle
    - The API stores the @type field using the class name

- Adding validation:
    - Symfony validator constraints on fields
    - Publish-only validation via groups (see Publishable doc)

- Combining annotations — a complete example:
    #[Silverback\Publishable]
    #[Silverback\Timestamped]
    #[ORM\Entity]
    #[ApiResource(mercure: true)]
    class Title extends AbstractComponent {
        use PublishableTrait, TimestampedTrait;
        #[ORM\Column(nullable: true)]
        #[Assert\NotBlank(groups: ['Title:published'])]
        public ?string $title = null;
    }

- What the API returns for a component (show example JSON with _metadata, @type, @id)
- Relation to the front-end: the @type value = the component name the Nuxt module uses to resolve CwaComponentTitle
