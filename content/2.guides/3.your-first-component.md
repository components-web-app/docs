## Step 1: Generate the API entity

In your API project:

```bash
php bin/console make:api-component Title --publishable
```

Then run the migration:

```bash
php bin/console doctrine:migrations:diff
php bin/console doctrine:migrations:migrate
```

See [`make:api-component`](/api/console-commands#make-api-component) for all available flags (`--timestamped`, `--uploadable`, etc.).

---

## Step 2: Scaffold the Vue component

In your Nuxt project:

```bash
npx cwa make:component
```

The CLI prompts for the component name, type, and behaviours, then writes `app/cwa/components/Title/Title.vue`. It also outputs the `php bin/console make:api-component` command for you — useful if you haven't run step 1 yet.

Open the generated file and fill in your template:

```vue
<template>
  <h1 class="text-4xl font-bold">
    {{ resource?.data?.title || 'No Title' }}
  </h1>
</template>
```

See [`cwa make:component`](/nuxt-module/building-your-ui/cwa-cli) for the full reference and the image/collection variants.

---

## Step 3: Add the admin tab

Create `app/cwa/components/Title/admin/Title.vue` — this is the editing form shown in the inline CMS:

```vue
<template>
  <CwaUiFormLabelWrapper label="Title">
    <CwaUiFormInput v-model="titleModel.model.value" />
  </CwaUiFormLabelWrapper>
</template>

<script setup lang="ts">
import { useCwaResourceManagerTab, useCwaResourceModel } from '#imports'

const { exposeMeta, iri } = useCwaResourceManagerTab({ name: 'Title' })
const titleModel = useCwaResourceModel<string>(iri, 'title')

defineExpose(exposeMeta)
</script>
```

---

## Step 4: Register in nuxt.config

```ts
// nuxt.config.ts
export default defineNuxtConfig({
  cwa: {
    resources: {
      Title: {
        name: 'Title',
        description: 'A heading or section title',
        defaultData: { title: 'New Title' }
      }
    }
  }
})
```

---

## Try it

Log in as admin, open edit mode, click **+** in a component group, find **Title**, add it, and type in the tab. Changes save automatically. Click **Publish** to make it live.
