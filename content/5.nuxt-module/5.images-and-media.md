<!--
COVER:
- When to use useCwaImageResource instead of useCwaResource:
    — Any component whose PHP entity uses the Uploadable annotation
    — Provides additional computed values for displaying the uploaded file

- useCwaImageResource usage:
    import { useCwaImageResource } from '#imports'
    const props = defineProps<IriProp>()
    const { resource, getResource, contentUrl, displayMedia, loaded, handleLoad, mediaObjects } = useCwaImageResource(props)
    defineExpose(getResource.exposeMeta)

- What it adds over useCwaResource:
    - contentUrl: the public URL of the primary uploaded file
    - displayMedia: boolean — true once the file URL is available and the image has loaded
    - loaded: ref<boolean> — becomes true when the <img> fires its load event
    - handleLoad(): call this on the <img> @load event
    - mediaObjects: the full _metadata.mediaObjects map (all variants)

- Displaying an image:
    <NuxtImg v-if="contentUrl" :src="contentUrl" @load="handleLoad" />
    <div v-else class="placeholder">No image</div>

- Imagine filter variants (srcset / responsive images):
    — Multiple versions of the image created by LiipImagine (thumbnail, hero, square, etc.)
    — Access a specific variant:
        const thumbnailUrl = mediaObjects.value?.file?.thumbnail?.contentUrl
    — Or use useCwaImage(iri, { imagineFilter: 'thumbnail' }) for a focused composable

- Transition while loading:
    <Transition name="fade">
        <NuxtImg v-show="displayMedia" :src="contentUrl" @load="handleLoad" />
    </Transition>

- Admin upload tab: app/cwa/components/Image/admin/Image.vue
    import { useCwaResourceManagerTab, useCwaResourceUpload } from '#imports'
    const props = defineProps<IriProp>()
    useCwaResourceManagerTab({ name: 'Image', order: 1 })
    const {
        filenameInputModel,
        handleInputChangeFile,
        handleInputDeleteFile,
    } = useCwaResourceUpload(props, 'file')  // 'file' = the UploadableField property name

    Template:
    <CwaUiFormFile
        :filename="filenameInputModel"
        @change="handleInputChangeFile"
        @delete="handleInputDeleteFile"
        label="Upload Image"
    />

- Video / PDF / other non-image uploads:
    — Use the same useCwaImageResource pattern
    — contentUrl is still the download URL
    — displayMedia is true when contentUrl is available (no image load event needed for non-images)
    — Render a <video :src="contentUrl" /> or <a :href="contentUrl"> as appropriate
