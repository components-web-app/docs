COVER:
- The /_cwa/settings admin page:
    — Only accessible to ROLE_ADMIN
    — All settings stored via the API and cached in the Pinia store
    — Changes take effect immediately (no deploy needed)

- Available settings (what the admin can configure):
    - Site name
    - Title separator (e.g. ' | ') and concat format (before/after)
    - Fallback title (used when a page has no title)
    - Canonical URL base (for hreflang/canonical meta)
    - Maintenance mode: on/off + custom message
    - Robots indexing: enabled/disabled + custom robots.txt content
    - Sitemap: enabled/disabled; the module auto-generates /sitemap.xml from routes

- useCwaSiteConfig composable:
    import { useCwaSiteConfig } from '#imports'
    const siteConfig = useCwaSiteConfig()
    siteConfig.siteName.value        // the site name
    siteConfig.maintenanceMode.value // boolean
    siteConfig.robotsEnabled.value

- Per-page SEO from the API:
    — Page.title → used as the <title> tag (concatenated with siteName)
    — Page.metaDescription → meta[name="description"]
    — PageData.title overrides Page.title for dynamic pages
    — PageData.metaDescription overrides for dynamic pages
    — All wired automatically by the module middleware into @nuxtjs/seo

- nuxt-seo integration (@nuxtjs/seo):
    — The module calls useSeoMeta() / useHead() with the resolved title and description
    — You can further override these in individual page templates:
        useHead({ title: 'Custom Title' })
    — Your override wins (set after the module's middleware)

- Open Graph / Twitter cards:
    — The module does not set OG tags automatically
    — Add them in your page templates or layouts:
        useSeoMeta({ ogImage: pageData?.data?.heroImageUrl })

- Maintenance mode:
    — When siteConfig.maintenanceMode.value is true, the module redirects all non-admin users
    — The redirect target is /_cwa/maintenance (a built-in page)
    — You can override this page by creating app/pages/_cwa/maintenance.vue

- Sitemap generation:
    — /sitemap.xml auto-generated from all Route resources
    — Only includes routes for published pages (not drafts)
    — Customise priority/changefreq in the route settings (/_cwa/routes admin)
    — Disable in settings if you manage a sitemap externally
