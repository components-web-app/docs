Use in **admin tab components** for uploadable components. Not for display components.

```ts
import { useCwaResourceManagerTab, useCwaResourceUpload } from '#imports'

const { exposeMeta, iri } = useCwaResourceManagerTab({ name: 'Upload' })
const {
  filenameInputModel,
  updating,
  fileExists,
  handleInputChangeFile,
  handleInputDeleteFile
} = useCwaResourceUpload(iri)

defineExpose(exposeMeta)
```

```vue
<template>
  <CwaUiFormFile
    v-model="filenameInputModel"
    label="Upload Image"
    :disabled="updating"
    :file-exists="fileExists"
    @change="handleInputChangeFile"
    @delete="handleInputDeleteFile"
  />
</template>
```

## Signature

```ts
useCwaResourceUpload(iri: Ref<string>)
```

Takes the `iri` ref from `useCwaResourceManagerTab`. The IRI identifies which resource to PATCH when a file is selected or deleted.

## Return values

| Return | Type | Purpose |
|--------|------|---------|
| `filenameInputModel` | `Ref<string \| undefined>` | Bind via `v-model` on `CwaUiFormFile` |
| `updating` | `Ref<boolean>` | `true` while an upload/delete is in progress |
| `fileExists` | `Ref<boolean>` | `true` when a file is already uploaded for this resource |
| `handleInputChangeFile` | `(event: Event) => Promise<void>` | Pass to `@change` on the file input |
| `handleInputDeleteFile` | `() => Promise<void>` | Pass to `@delete` on the file input |

## How it works

**`handleInputChangeFile`** — reads the selected file, encodes it as base64, and PATCHes the API. Sets `updating.value = true` during the upload. On success the resource store updates automatically.

**`handleInputDeleteFile`** — sends a PATCH with `null` for the file field. The API deletes the stored file and all Imagine variants.

**`filenameInputModel`** — bind with `v-model` on `CwaUiFormFile`. Shows the current filename when a file is uploaded.

## `CwaUiFormFile` props

```vue
<CwaUiFormFile
  v-model="filenameInputModel"   <!-- current filename for display -->
  label="Image"
  :disabled="updating"           <!-- disable input while uploading -->
  :file-exists="fileExists"      <!-- shows delete button when true -->
  accept="image/*"               <!-- optional MIME restriction -->
  @change="handleInputChangeFile"
  @delete="handleInputDeleteFile"
/>
```

## Complete admin upload tab example

```vue
<!-- app/cwa/components/HeroSection/admin/ImageTab.vue -->
<template>
  <CwaUiFormFile
    v-model="filenameInputModel"
    label="Upload Image"
    :disabled="updating"
    :file-exists="fileExists"
    @change="handleInputChangeFile"
    @delete="handleInputDeleteFile"
  />
</template>

<script setup lang="ts">
import { useCwaResourceManagerTab, useCwaResourceUpload } from '#imports'

const { exposeMeta, iri } = useCwaResourceManagerTab({ name: 'Upload' })
const { filenameInputModel, updating, fileExists, handleInputChangeFile, handleInputDeleteFile }
  = useCwaResourceUpload(iri)

defineExpose(exposeMeta)
</script>
```
